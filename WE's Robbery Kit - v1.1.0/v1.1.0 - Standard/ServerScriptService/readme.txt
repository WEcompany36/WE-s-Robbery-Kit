WE's Robbery Kit - v1.1.0 Standard

How to set up
1. Insert Leaderboard.lua in ServerScriptService in Studio
2. Add the Attribute CurrencyName as a String
3. Add the Attribute DefaultValue as a Number
4. Add the Attribute FolderName as a String
5. Change to whatever you want

6. Add RobValues.lua in ServerScriptService
7. Add the Attrubute CashCollectedName as a String
8. Add the Attrubute Robbing as a String
9. Change to whatever you want
