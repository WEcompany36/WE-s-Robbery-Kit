WE's Robbery Kit - v1.1.0 Standard

How to set up
1. Add ReplicatedStorage.rbxm to your game inside of ReplicatedStorage, or add RemoteEvent: HideRobberyGUI and RemoteEvent: ShowRobberyGUI to your game
2. Add the StandaloneGui.rmxm to your game inside of StarterGui. There are some scripts in there that you will need also. 
3. Follow the other ReadMe.txt files in the other folders
